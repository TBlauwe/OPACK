 \hypertarget{index_autotoc_md1}{}\doxysection{OPACK}\label{index_autotoc_md1}
(Last known as {\bfseries{Dynamo}})

{\bfseries{OPACK}} is a C++ library to simulate cognitive agents. The main goal is to allow quick construction of an agent model tailored to your needs. An agent model in OPACK is defined as follow \+:


\begin{DoxyItemize}
\item {\bfseries{O}} \+: set of operations manipulating {\bfseries{PACK}}
\item {\bfseries{PACK}} \+: set of data specific for an agent
\item {\bfseries{P}} \+: set of current percepts
\item {\bfseries{A}} \+: set of current actions
\item {\bfseries{C}} \+: set of caracteristics (stable, e.\+g personality traits, or dynamic, e.\+g stress, emotion, etc.)
\item {\bfseries{K}} \+: set of knowledge.
\end{DoxyItemize}

We provide tools to facilitate the construction of an OPACK that suit your needs. While a whole virtual environment can be created using this tool, here we suppose you already have one.
